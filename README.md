# UPI Fraud Detection System

A comprehensive full-stack web application for real-time UPI transaction fraud detection using machine learning.

## Project Structure

```
detect/
├── backend/          # Node.js/Express API server
├── frontend/         # React.js web application
├── ml-api/          # Python Flask ML inference API
├── data/            # Sample datasets and model files
└── docs/            # Documentation
```

## Tech Stack

- **Frontend**: React.js, Tailwind CSS, Chart.js
- **Backend**: Node.js, Express.js, MongoDB
- **ML**: Python, Scikit-learn, Flask
- **Authentication**: JWT, bcrypt
- **Database**: MongoDB

## Features

- Real-time fraud detection
- User authentication and authorization
- Transaction management
- Analytics dashboard
- Alert system
- Search and filtering

## Quick Start

1. Install dependencies for all components
2. Set up MongoDB database
3. Train and deploy ML model
4. Start all services

See individual component READMEs for detailed setup instructions.
